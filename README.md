# PROJECT CRUZ

This is a static website created using HTML, CSS, and Javascript. 
A consultation firm website created for client to book hassle free wherever they are and whenever they like, all from their own comfort.

index.html: The main landing page for Project Cruz.
about.html: Information regarding the firm's mission and background.
services.html: A detailed list of the consultation services offered by the firm.
contact.html: Contact information and a form for clients to reach out.
case-studiews.html: The primary page for inquiries and booking appointments.
